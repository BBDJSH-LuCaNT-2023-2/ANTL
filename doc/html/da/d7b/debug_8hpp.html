Detailed Description

Macros and functions useful for debugging.

Author
John Lees-Miller You can turn off errors and "info" messages by defining NDEBUG. This is usually done in the makefile, and also disables C assertions. For gcc, you would do this by passing the -DNDEBUG option on the command line.
