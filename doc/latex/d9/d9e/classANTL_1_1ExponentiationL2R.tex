\subsection{Detailed Description}
\subsubsection*{template$<$class T$>$class A\-N\-T\-L\-::\-Exponentiation\-L2\-R$<$ T $>$}

class for left-\/to-\/right N\-A\-F exponentiation with N\-A\-F representation also computed left-\/to-\/right 

\begin{DoxyRemark}{Remarks}
This concrete class defines a method for left-\/to-\/right non-\/adjacent form exponentiation. The method computes the N\-A\-F representation of the exponent using a left-\/to-\/right method, meanwhile applying it to the base left-\/to-\/right. The base type is templated, and must have the following functions defined\-:
\begin{DoxyItemize}
\item assign(\-T \&\-C, const T \&\-A)\-: sets C = A
\item mul(\-T \&\-C, const T \&\-A, const T \&\-B)\-: computes C = A\-B
\item sqr(\-T \&\-C, const T \&\-A)\-: computes C = A\-A
\item inv(\-T \&\-C, const T \&\-A)\-: computes C = inverse of A 
\end{DoxyItemize}
\end{DoxyRemark}
