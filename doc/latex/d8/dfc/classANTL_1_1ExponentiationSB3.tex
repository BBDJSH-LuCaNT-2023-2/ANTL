\subsubsection*{template$<$class T$>$class A\-N\-T\-L\-::\-Exponentiation\-S\-B3$<$ T $>$}

class for left-\/to-\/right S\-B3 exponentiation 

\begin{DoxyRemark}{Remarks}
This concrete class defines a method for left-\/to-\/right signed base 3 exponentiation. The method first computes the signed ternary representation of the exponent using the basic right-\/to-\/left method, and then applies it to the base left-\/to-\/right. The base type is templated, and must have the following functions defined\-:
\begin{DoxyItemize}
\item assign(\-T \&\-C, const T \&\-A)\-: sets C = A
\item mul(\-T \&\-C, const T \&\-A, const T \&\-B)\-: computes C = A\-B
\item inv(\-T \&\-C, const T \&\-A)\-: computes C = inverse of A
\item cub(\-T \&\-C, const T \&\-A)\-: computes C = A\-A\-A 
\end{DoxyItemize}
\end{DoxyRemark}
